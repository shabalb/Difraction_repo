# Дифракция цилиндрической звуковой волны на упругом цилиндре с дискретно-слоистым покрытием

## Постановка задачи


Рассматривается бесконечный однородный изотропный упругий цилиндр радиуса $r_0$ с плотностью $\rho_0$ и упругими постоянными Ламе $\lambda_0, \mu_0$. Цилиндр окружён $N$ тонкими коаксиальными упругими слоями радиусами $r_j$ ($j = 1, \dots, N$), каждый из которых характеризуется плотностью $\rho_j$ и параметрами $\lambda_j, \mu_j$. Внешняя среда — идеальная жидкость с плотностью $\rho$ и скоростью звука $c$.

В цилиндрической системе координат $(r, \varphi, z)$, где ось $z$ совпадает с осью цилиндра, на тело падает цилиндрическая звуковая волна, излучаемая нитевидным источником, параллельным оси $z$. Источник расположен в точке $(r_s, \varphi_s)$. Задача считается двумерной.

### Падающее поле

Потенциал скоростей падающей волны имеет вид:

$$
\Psi_0 = A H_0^{(1)}(kR) e^{-i\omega t},
$$

где $R = \sqrt{r^2 + r_s^2 - 2rr_s \cos(\varphi - \varphi_s)}$ — расстояние от источника до точки наблюдения, $k = \omega / c$ — волновое число в жидкости, $H_0^{(1)}$ — функция Ханкеля первого рода нулевого порядка.

Результирующее поле в жидкости:

$$
\Psi = \Psi_0 + \Psi_s,
$$

где $\Psi_s$ — потенциал рассеянной волны, удовлетворяющий уравнению Гельмгольца:

$$
\Delta \Psi_s + k^2 \Psi_s = 0.
$$

Скорость и давление в жидкости выражаются через потенциал:

$$
\mathbf{v} = \nabla \Psi, \qquad p = i \rho \omega \Psi.
$$

## Решение задачи

### Разложение падающей волны в ряд Фурье

Падающая волна разлагается по цилиндрическим гармоникам:

$$
\Psi_0(r, \varphi) = A \sum_{n=-\infty}^{\infty} J_n(k r_s) H_n^{(1)}(k r) \, e^{i n (\varphi - \varphi_s)}, \quad r > r_s,
$$

где $J_n$ — функция Бесселя, $H_n^{(1)}$ — функция Ханкеля первого рода.

Для рассеянной волны принимается представление:

$$
\Psi_s(r, \varphi) = \sum_{n=-\infty}^{\infty} A_n H_n^{(1)}(k r) \, e^{i n (\varphi - \varphi_s)},
$$

где $A_n$ — неизвестные коэффициенты, подлежащие определению.

### Поля в упругих слоях

В каждом упругом слое $j$ ($j = 0$ — центральный цилиндр, $j = 1, \dots, N$ — покрытия) вектор смещения $\mathbf{u}^{(j)}$ представляется через скалярный и векторный потенциалы:

$$
\mathbf{u}^{(j)} = \nabla \Psi^{(j)} + \nabla \times \Phi^{(j)}.
$$

В двумерном случае векторный потенциал имеет только $z$-компоненту: $\Phi^{(j)} = \Phi^{(j)}(r, \varphi) \mathbf{e}_z$.

Потенциалы удовлетворяют уравнениям Гельмгольца:

$$
\Delta \Psi^{(j)} + (k_l^{(j)})^2 \Psi^{(j)} = 0,
$$
$$
\Delta \Phi^{(j)} + (k_t^{(j)})^2 \Phi^{(j)} = 0,
$$

где $k_l^{(j)} = \omega / c_l^{(j)}$, $k_t^{(j)} = \omega / c_t^{(j)}$ — волновые числа продольных и поперечных волн, а скорости:

$$
c_l^{(j)} = \sqrt{\frac{\lambda_j + 2\mu_j}{\rho_j}}, \qquad c_t^{(j)} = \sqrt{\frac{\mu_j}{\rho_j}}.
$$

Решения для потенциалов в слое записываются в виде:

$$
\Psi^{(j)}(r, \varphi) = \sum_{n} \left[ B_{1n}^{(j)} J_n(k_l^{(j)} r) + B_{2n}^{(j)} N_n(k_l^{(j)} r) \right] e^{i n (\varphi - \varphi_s)},
$$

$$
\Phi^{(j)}(r, \varphi) = \sum_{n} \left[ C_{1n}^{(j)} J_n(k_t^{(j)} r) + C_{2n}^{(j)} N_n(k_t^{(j)} r) \right] e^{i n (\varphi - \varphi_s)},
$$

где $N_n$ — функция Неймана. Для центрального цилиндра ($j=0$) в силу ограниченности поля на оси $B_{2n}^{(0)} = C_{2n}^{(0)} = 0$.

### Граничные условия

1. **На внешней поверхности ($r = r_N$)**:
   - равенство нормальных скоростей жидкости и упругой среды: $-i\omega u_r^{(N)} = v_r$,
   - равенство нормального напряжения и давления: $\sigma_{rr}^{(N)} = -p$,
   - отсутствие касательного напряжения: $\sigma_{r\theta}^{(N)} = 0$.

2. **На границах между слоями ($r = r_j$, $j = 1, \dots, N-1$)**:
   - непрерывность смещений: $u_r^{(j)} = u_r^{(j+1)}$, $u_\varphi^{(j)} = u_\varphi^{(j+1)}$,
   - непрерывность напряжений: $\sigma_{rr}^{(j)} = \sigma_{rr}^{(j+1)}$, $\sigma_{r\theta}^{(j)} = \sigma_{r\theta}^{(j+1)}$.

Компоненты напряжений вычисляются через потенциалы смещения по формулам:

$$
\sigma_{rr}^{(j)} = \lambda_j \left( \frac{\partial u_r}{\partial r} + \frac{1}{r}\frac{\partial u_\varphi}{\partial \varphi} + \frac{u_r}{r} \right) + 2\mu_j \frac{\partial u_r}{\partial r},
$$

$$
\sigma_{r\theta}^{(j)} = \mu_j \left( \frac{1}{r}\frac{\partial u_r}{\partial \varphi} + \frac{\partial u_\varphi}{\partial r} - \frac{u_\varphi}{r} \right).
$$

А компоненты смещения через потенциалы:

$$
u_r^{(j)} = \frac{\partial \Psi^{(j)}}{\partial r} + \frac{1}{r}\frac{\partial \Phi^{(j)}}{\partial \varphi},
$$

$$
u_\varphi^{(j)} = \frac{1}{r}\frac{\partial \Psi^{(j)}}{\partial \varphi} - \frac{\partial \Phi^{(j)}}{\partial r}.
$$

## Матричный метод решения

Для уменьшения размера системы уравнений применяется метод передаточных матриц.

### Вектор состояния

В каждом слое вводится вектор смещения-напряжения для каждой гармоники $n$:

$$
\mathbf{S}_n^{(j)}(r) = \left[ u_{rn}^{(j)}(r),\; u_{\varphi n}^{(j)}(r),\; \sigma_{rr n}^{(j)}(r),\; \sigma_{r\theta n}^{(j)}(r) \right]^T.
$$

Этот вектор связан с вектором коэффициентов потенциалов $\mathbf{K}_n^{(j)} = [B_{1n}^{(j)}, B_{2n}^{(j)}, C_{1n}^{(j)}, C_{2n}^{(j)}]^T$ через матрицу $T_n^{(j)}(r)$ размера $4 \times 4$:

$$
\mathbf{S}_n^{(j)}(r) = T_n^{(j)}(r) \, \mathbf{K}_n^{(j)}.
$$

Элементы матрицы $T_n^{(j)}(r)$ вычисляются аналитически через функции Бесселя, Неймана и их производные (полные выражения приведены в статье).

### Переход через слой

Используя связь векторов на внутренней и внешней границах слоя, получаем:

$$
\mathbf{S}_n^{(j)}(r_{j-1}) = P_n^{(j)} \, \mathbf{S}_n^{(j)}(r_j),
$$

где

$$
P_n^{(j)} = T_n^{(j)}(r_{j-1}) \cdot \left[ T_n^{(j)}(r_j) \right]^{-1}.
$$

Последовательно применяя этот переход для всех слоёв, связываем вектор состояния на поверхности центрального цилиндра ($r = r_0$) с вектором на внешней границе ($r = r_N$):

$$
\mathbf{S}_n^{(N)}(r_N) = P_n \, \mathbf{S}_n^{(0)}(r_0),
$$

где $P_n = P_n^{(N)} P_n^{(N-1)} \dots P_n^{(1)}$ — полная матрица перехода.

### Формирование системы линейных уравнений

На внешней границе ($r = r_N$) граничные условия выражаются через компоненты вектора $\mathbf{S}_n^{(N)}(r_N)$:

- $u_{rn}^{(N)} = \frac{1}{-i\omega} \frac{\partial}{\partial r} \left[ A H_n^{(1)}(k r_s) J_n(k r) + A_n H_n^{(1)}(k r) \right]_{r = r_N}$,
- $\sigma_{rr n}^{(N)} = -i\rho\omega \left[ A H_n^{(1)}(k r_s) J_n(k r_N) + A_n H_n^{(1)}(k r_N) \right]$,
- $\sigma_{r\theta n}^{(N)} = 0$.

Подстановка этих выражений в матричное соотношение для $P_n$ с учётом ограниченности поля в центре ($B_{2n}^{(0)} = C_{2n}^{(0)} = 0$) даёт систему трёх линейных уравнений относительно неизвестных $A_n$, $B_{1n}^{(0)}$, $C_{1n}^{(0)}$ для каждого $n$.

Система решается численно для конечного набора гармоник $n = -N_{\max}, \dots, N_{\max}$.

## Численный эксперимент

### Параметры модели

- Центральный цилиндр: алюминий  
  $r_0 = 2$ м (в условных единицах),  
  $\rho_0 = 2.7 \cdot 10^3$ кг/м³,  
  $\lambda_0 = 5.3 \cdot 10^{10}$ Н/м²,  
  $\mu_0 = 2.6 \cdot 10^{10}$ Н/м².

- Покрытие (один слой для примера):  
  $\rho_1 = 1$ кг/м³,  
  $\lambda_1 = 1$ Н/м²,  
  $\mu_1 = 1$ Н/м².

- Внешняя среда: вода  
  $\rho = 1000$ кг/м³,  
  $c = 1485$ м/с.

- Частота и волновое число выбираются так, чтобы $k r_0 \sim 1$ (резонансный диапазон).

### Результаты

На рисунке ниже показаны диаграммы направленности рассеянного поля для двух положений источника:

- **Удалённый источник** ($r_s = 50 \cdot r_N$) — диаграмма практически совпадает с рассеянием плоской волны.
- **Близкий источник** ($r_s = 2 \cdot r_N$) — наблюдаются искажения диаграммы, характерные для цилиндрической волны.

Диаграммы нормированы на максимум для исключения эффекта ослабления амплитуды с расстоянием.

### Выводы

- Получено аналитико-численное решение задачи дифракции цилиндрической волны на дискретно-слоистом упругом цилиндре.
- Для удалённого источника решение переходит в известный случай рассеяния плоской волны, что подтверждает корректность модели.
- Варьируя параметры слоёв покрытия, можно управлять характеристиками отражённого поля (формой диаграммы, уровнем боковых лепестков и т.д.).

## Структура репозитория

- `src/` — исходные коды программы (язык: Python/Matlab/Julia — по вашему выбору).
- `matrices/` — функции для построения матриц $T_n^{(j)}(r)$ и $P_n^{(j)}$.
- `solve/` — решатель системы линейных уравнений для каждого $n$.
- `examples/` — скрипты для воспроизведения численных экспериментов.
- `figures/` — сгенерированные графики.
- `docs/` — дополнительные материалы (например, полный вывод матричных элементов).
