# Рассеяние плоской звуковой волны цилиндром с секторным покрытием

Решение задачи рассеяния плоской звуковой волны бесконечным упругим цилиндром, покрытие которого разбито по угловой координате на однородные секторы. Такая модель позволяет приближать непрерывно неоднородное по углу покрытие набором областей с постоянными упругими параметрами. Численное решение реализовано в plane_to_seg.m.

## Постановка задачи

Рассматривается упругий цилиндр радиуса $r_0$, окружённый покрытием с внешним радиусом $r_1$. Покрытие состоит из $N$ секторов. Для каждого сектора задаются плотность $\rho_j$ и постоянные Ламе $\lambda_j$, $\mu_j$. Внешняя среда — идеальная жидкость с плотностью $\rho$ и скоростью звука $c$.

На цилиндр под углом $\varphi_s$ падает плоская гармоническая волна. Требуется определить коэффициенты рассеянного поля и построить его диаграмму направленности.

## Метод решения

Акустическое поле во внешней жидкости представляется суммой падающей и рассеянной волн. Падающая волна раскладывается по функциям Бесселя, а рассеянная — по функциям Ханкеля первого рода:

$$
\Psi_s(r,\varphi)=\sum_{n=-\infty}^{\infty} A_n H_n^{(1)}(kr)
e^{in(\varphi-\varphi_s)}.
$$

В центральном цилиндре и каждом секторе упругое смещение задаётся через продольный и поперечный потенциалы:

$$
\mathbf u^{(j)}=grad\Psi^{(j)}+rot\Phi^{(j)}.
$$

Потенциалы раскладываются по цилиндрическим гармоникам с функциями Бесселя и Неймана. В центральном цилиндре функции Неймана исключаются из условия ограниченности решения при $r=0$.

Для каждой гармоники вводятся матрицы, связывающие коэффициенты потенциалов с компонентами смещения и напряжения. Коэффициенты определяются из следующих граничных условий:

* на границе цилиндра и покрытия $r=r_0$ непрерывны $u_r$, $u_\varphi$, $\sigma_{rr}$ и $\sigma_{r\varphi}$;
* на внешней поверхности $r=r_1$ согласуются нормальные скорости и давления, а касательное напряжение равно нулю;
* на радиальных границах соседних секторов непрерывны $u_r$, $u_\varphi$, $\sigma_{\varphi\varphi}$ и $\sigma_{r\varphi}$.

Условия на дугах проецируются на гармонический базис отдельно на угловом интервале каждого сектора. Для этого используется интеграл

$$
I_{mn}^{(j)}=
\int_{\varphi_j}^{\varphi_{j+1}}
e^{i(n-m)(\varphi-\varphi_s)},d\varphi.
$$

Такое проецирование учитывает, что экспоненты не ортогональны на отдельном секторе. Условия на радиальных границах секторов выполняются методом коллокации; точки коллокации выбираются в корнях многочленов Лежандра.

После усечения рядов до $|n|\leq N_m$ все условия объединяются в комплексную систему линейных алгебраических уравнений относительно:

* коэффициентов рассеянной волны $A_n$;
* коэффициентов продольного и поперечного потенциалов центрального цилиндра;
* четырёх наборов коэффициентов потенциалов каждого сектора покрытия.
